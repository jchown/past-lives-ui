# Past Lives UI

Front end code to show ["past lives"](https://www.datasmelter.com/history/past-lives/).

Built using Yarn, Vue 3 and Vuetify.

Back end code at [https://github.com/jchown/past-lives-api](https://github.com/jchown/past-lives-api).

## Development

To run locally use `yarn dev`.

To deploy push to the `main` branch.
